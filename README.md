# 쇼핑몰 프로젝트

개인 공부용 프로젝트 입니다.

사용 책 : [스프링 부트 쇼핑몰 프로젝트 with JPA](http://www.yes24.com/Product/Goods/103453774)

## 🚩 목표

1. 협업 연습을 위한 깃허브 설정 연습
2. 쇼핑몰 프로젝트를 앞두고서 프로젝트 연습
3. JPA의 연관 관계 연습
4. 결제 주문 시스템의 이해
5. 단순히 책을 따라하는 것이 아닌 내가 사용하고 싶은 기술 or 코딩 방식 사용
